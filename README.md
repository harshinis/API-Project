# API-Project
AJAX API - Quiz app

Developing this Quiz Application helped understand
1) How to connect to Trivia database and return JSON data for web pages
2) How to make AJAX requests and retrieve response data

This simple Quiz Applications helps learn how to use a web API to make an AJAX request for data.  Retrieve the response data and make use of it to build a mini trivia game using JavaScript.

Used AJAX, JSON and JavaScript to explore fun ways to use APIs.   AJAX allows web developers a way to make http requests in the background to help improve the web users experience by providing a seamless way to load new data and update the page elements.
